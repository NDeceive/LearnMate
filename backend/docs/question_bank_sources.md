# 题库来源说明

## system_seed

`system_seed` 是计智引擎项目自建基础题库来源，用于 Phase 3 已完成的 `question_bank` 基础选择题数据。

## interview_grill_adapted

`interview_grill_adapted` 参考 interview-grill 的 408 学习训练结构，重新整理和改编为本项目可用的开放追问素材与后续可扩展选择题素材。

当前开放题库定位为 AI 导师讲解、追问、错因分析和 checkpoints 参考，不替代 Phase 3 已有基础选择题。

## 引入边界

本项目未引入 interview-grill 前端代码。

本项目未引入 Electron。

本项目未整包复制 interview-grill 代码，仅参考其学习训练结构并面向计智引擎 408 学习场景重新组织题库内容。
