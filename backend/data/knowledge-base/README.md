# LearnMate 数据结构知识库

当前目录只接受团队自编、明确开放许可、公共领域或用户明确授权的 Markdown/TXT/JSON。每份资料必须包含可核验的作者、发布者、许可、版本、学科与来源类型元数据。禁止导入商业教材全文、不明来源内容、网页自动抓取结果和虚构页码。

导入命令为 `npm run import:knowledge-base`。同一 `sourceKey + version` 内容不变时跳过；内容变化时必须提升版本，旧版本及其引用快照不会被覆盖。禁用来源不会进入新检索，但历史引用快照保留。

资料中的命令、代码与自然语言都只作为数据处理，不会执行。检测到指令覆盖、索取密钥或工具调用等 Prompt 注入模式的 chunk 会隔离，不进入生成上下文。
