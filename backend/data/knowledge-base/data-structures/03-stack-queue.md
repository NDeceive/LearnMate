# 栈与队列
## 栈定义
栈是只在栈顶执行插入和删除的后进先出结构。push 加入元素，pop 删除栈顶，top 或 peek 只读取栈顶，空栈操作必须有明确错误语义。
## 栈实现
顺序栈用动态数组维护栈顶下标，链式栈把表头作为栈顶。两者正常 push 和 pop 都为 O(1)，顺序栈扩容的单次最坏代价为 O(n)。
## 栈应用
函数调用栈保存返回位置与局部状态；表达式求值用栈匹配运算符优先级；深度优先搜索可用显式栈替代递归，便于控制深度。
## 队列定义
队列按先进先出规则工作，队尾 enqueue，队头 dequeue。队列适合表示等待顺序、广度优先搜索前沿和生产者消费者缓冲区。
## 循环队列
循环队列让头尾下标通过模运算回绕，避免顺序队列出队后不断搬移元素。区分队空与队满可预留一个槽位，或额外维护元素数量。
## 链式队列
链式队列同时保存头指针和尾指针，入队连接尾部，出队删除头部。删除最后一个元素后必须把尾指针同步置空。
## 双端队列
双端队列允许两端插入和删除，可实现滑动窗口最值中的单调队列。它不等同于优先队列，元素出队依据位置而不是键的优先级。
## 示例
括号匹配从左向右扫描，遇左括号入栈，遇右括号检查并弹出匹配类型。结束后栈也必须为空，否则仍存在未闭合括号。
## 复杂度
栈和队列的基本端点操作通常为 O(1)。若用普通数组每次删除首元素并整体移动，出队会退化为 O(n)，循环下标可避免该问题。
## 常见错误
循环队列常见错误包括 front 与 rear 含义不一致、模运算遗漏以及容量计算差一。栈算法常见错误是未检查空栈就读取栈顶。
